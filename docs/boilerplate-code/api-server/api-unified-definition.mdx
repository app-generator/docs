# API Unified Definition

import SubHeading from "@site/src/components/SubHeading";

<SubHeading color="#25c2a0">The common definition used in all AppSeed products that includes an API Service</SubHeading>

This page describes the **unified definition** used by all API servers provided by AppSeed. This internal standard is used by all full-stack products, despite the UI or backend technology.

> API Servers aligned to use it:

- [API Server Django](django.mdx) - free product
- [API Server Flask](flask.mdx) - free product
- [API Server Node JS](node-js.mdx) - free product
- [API Server Node JS Mongo](https://github.com/app-generator/api-server-nodejs-mongo) - free product


## API Information

> Interface descriptor - [POSTMAN collection format](https://github.com/app-generator/api-unified-definition/blob/main/api.postman_collection.json)

- USERS API:
  - `/api/users/register`: create a new user
  - `/api/users/login`: authenticate an existing user
  - `/api/users/logout`: delete the associated JWT token
  - `/api/users/checkSession`: check an existing JWT Token for validity
  - `/api/users/edit` - edit the information associated with a registered user
