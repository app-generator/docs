# [Azia Dashboard PRO](https://www.bootstrapdash.com/product/azia-admin?ref=23)

<SubHeading>Premium Bootstrap Dashboard template crafted by BootstrapDash</SubHeading>

**Azia Dashboard** is a Bootstrap 4 admin & dashboard template featuring components that are built to fit perfectly with each other. 
It offers a clean interface that can be easily adapted to fit your vision and branding. 
**Azia** features many essential utility classes that can help cut down CSS bloat and modify CSS components instantly.
