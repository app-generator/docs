# [Paper Dashboard](https://www.creative-tim.com/product/paper-dashboard-2?AFFILIATE=128200)

<SubHeading>Open-source Bootstrap 4 dashboard template crafted by Creative-Tim</SubHeading>

**Paper Dashboard 2** is a beautiful Bootstrap 4 admin dashboard with a large number of components, designed to look neat and organized. 
If you are looking for a tool to manage and visualize data about your business, this dashboard is the thing for you. 
It combines colors that are easy on the eye, spacious cards, beautiful typography, and graphics. 
Paper Dashboard 2 comes packed with all plugins that you might need inside a project and documentation on how to get started. It is light and easy to use, and also very powerful.
