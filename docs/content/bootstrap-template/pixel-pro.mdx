**Pixel Pro** is a premium [Bootstrap 5](https://www.admin-dashboards.com/bootstrap-5-templates/) UI Kit (no jQuery) featuring over 1000 components, 50+ sections, and 35 example pages including a fully-fledged user dashboard. 
**Pixel** is compliant with the latest UI design **accessibility standards** and passes the WAVE evaluation tool and the Achecker tool as well.
