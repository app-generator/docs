# [Now UI Dashboard](https://www.creative-tim.com/product/now-ui-dashboard?AFFILIATE=128200)

<SubHeading>Open-source Bootstrap Dashboard designed by Invision crafted by Creative-Tim</SubHeading>

**Now UI Dashboard** is a responsive Bootstrap 4 kit provided for free by `Invision` and `Creative-Tim`. It combines colors that are easy on the eye, spacious cards, beautiful typography, and graphics.

`Now UI Dashboard` comes packed with all plugins that you might need inside a project and documentation on how to get started. 
It is light and easy to use, and also very powerful. This makes starting a new project very simple. 
It also provides benefits if you are already working on a Bootstrap 4 project; you can just import the Now UI Dashboard style over it.
